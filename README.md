# Crop-Recommendation-Using-Machine-Learning-and-Deep-Learning

## Overview
This project predicts the most suitable crop based on soil nutrients and environmental conditions using Machine Learning and Deep Learning techniques.

## Dataset Features
- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH
- Rainfall

## Machine Learning Models
- Decision Tree
- Random Forest
- Naive Bayes
- KNN
- SVM
- Logistic Regression
- Gradient Boosting
- Ensemble (RF + NB + GB)

## Deep Learning Models
- ANN
- CNN
- LSTM
- CNN-LSTM
- TabNet

## Results
- TabNet: 99.55%
- Naive Bayes: 99.55%
- Random Forest: 99.32%
- Ensemble (RF+NB+GB): 99.32%
- Gradient Boosting: 98.18%
